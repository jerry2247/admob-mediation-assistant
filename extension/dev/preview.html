what's the difference between bidding and waterfall?

Bidding runs a real-time auction across your sources for each request — the highest bid wins. Waterfall calls sources in a fixed order by the eCPM you set, top to bottom.

Bidding usually maximizes competition; waterfall gives you manual control. Most groups today lead with bidding and add a few waterfall sources.
